CatViz Help Page

This page opens up on program's initialization if the variable OPEN_HELP_PAGE_ON_STARTUP is set as true in file config.js

The canvas starts with a border, lines, these lines' coordinates for orientation and an inner space inside which objects can be create and dragged
Such graphical assistances can be disabled in config.js

Create object: holding SHIFT key, click on the canvas inside the inner area
Create morphism: holding SHIFT key, click on the source object, hold the mouse down and release it on the target object
Create endomorphism: holding SHIFT key, click on the object and release it on the same object (if a double click occurs in up to 200 milliseconds, the endomorphism's creation is undone)

Select object: click on the object (while selected, its color is different)
Select morphism: click on the handle of the morphism (while selected, its color and its handler's color are different)
Select endomorphism: click on the handle of the endomorphism
Select multiple elements individually: holding CTRL key, click on each element
Select multiple elements with selection rectangle: click on the canvas, drag the mouse and release

Drag object: click on the object to select it, hold the mouse down and move it, release it to stop dragging
Drag morphism: click on the morphism's handle to select it, hold the mouse down and move it, release it to stop dragging
Drag endomorphism: click on the endomorphism's handle to select it, hold the mouse down and move it, release it to stop dragging
Drag multiple elements: click on some selected element, hold the mouse down and move it, release it to stop dragging (not implemented yet)

Select all: CTRL + A
"Deselect" all: CTRL + SHIFT + A
Copy selected elements: CTRL + C
Open file with saved states: CTRL + O (in development)
Create file with saved states: CTRL + P (in development)
Copy current state: CTRL + S
Paste copied elements: CTRL + V
Cut selected: CTRL + X
Redo last action: Ctrl + Y (in development)
Undo last action: Ctrl + Z (in development)

Delete selected: DEL (in development)

Open help page: F1
Open developers' page: F2
Open console in case of invisible faults: See this link
Debugging:
 Read information about the data structures of objects and morphisms: SPACE
 Read information about the graphical structures of objects and morphisms: CTRL+SPACE
 Read keyboard and mouse state machine's state: , (comma)
 Read selected elements: . (dot)
 Read state stack: ; (semicolon)

 Message body: steps necessary for the bug to manifest itself, separated by commas, followed by colon explaining what is the bug, followed by an optional solution in parentheses
 Example: create object, create object, create morphism: something bad happens (change line n in file.js)

For some of these actions, a second menu with specific fields opens up
Depending on the action, fields can be manipulated in this second menu before their actions are performed
In this second menu, fields can be reset to default if the RESET button is pressed
This second menu performs the actions if its OK button or the ENTER key are pressed
This second menu can be closed without its respective action taking effect if the QUIT button or ESC key are pressed.
The second menu is only executed if all its fields are filled
One can pass through the fields that must be filled using the TAB key
The value of the field type for morphisms can also be chosen with the numbers from 1 to 5

Change type of next created morphisms: Q (morphism, is the default), W (monomorphism), E (epimorphism), R (monoepimorphism), T (isomorphism)
Draw freely: holding TAB key, click and drag mouse while you want to draw (cursor will change shape while drawing)
Change drawing brush color: 0 to 9 (1 to 6 for rainbow colors and 7 to 0 for 4 monochromatic tones, default is 7)
Increase/decrease drawing brush size: +/- (1 to 72, default is 4)
Erase last drawing: BACKSPACE
Erase all drawings: CTRL + BACKSPACE

While the canvas is out of focus, a gray rectangle will fill it out, indicating that it is necessary to reposition the focus on it, this can be done simply by clicking on it
The focus is lost when opening the console, clicking and/or interacting with the console, clicking on the browser address and search bars, when an alert is being shown, etc
While ALT + TAB is being held, the focus is lost until the user goes back to the page
Attention: When the focus is lost, the selection rectangle is deleted and the keyboard and mouse state machine's state goes to "idle" or "idle with element(s) selected" (which means it could be used to reset when you notice a command is not working)

The config.js file contains user configurable values ​​(caution is necessary when modifying them)

The locale.js file allows the localization of the application (currently translated to de_DE, en_US, es_ES, fr_FR, it_IT, pt_BR, set by the LANGUAGE variable in config.js, default is en_US)

 Example: STR_OBJECT = "OBJECT"

The help page can also be corrected or expanded
Sending help page corrections:
 Send e-mail to rma@inf.ufrgs.br
 Subject: CatViz help page correction
 Message body: text snippet inside quotes, hyphen, corrected snippet inside quotes
 Example: "This is a wrong sentence" - "This is a correct sentence"
Sending help page expansions:
 Send e-mail to rma@inf.ufrgs.br
 Subject: CatViz help page expansion
 Message body: copy of the example.txt file with new language name in English, locale code, language name in native language, translation to "CatViz Help Page" and "Back to the beginning"
 Example: see the link
Keep a formal language and use equivalent conjugation tenses, such as infinitive in the names of the commands
If necessary, use a text editor that allows for codification change, such as UTF-8 and maintain the HTML tags
Incomplete help page expansions will not be accepted

Warning for developers: The file utils.js contains methods added to the Array and Math classes, so problems can arise in the future if the language is modified (to mitigate this, before each method there is an if to check if it is overriding an existing one)

Página de ajuda do CatViz

Esta página abre na inicialização do programa se a variável OPEN_HELP_PAGE_ON_STARTUP estiver configurada como true no arquivo config.js

O canvas começa com uma borda, linhas, coordenadas destas linhas para orientação a um espaço interno dentro do qual objetos podem ser criados e arrastados
Tais assistências gráficas podem ser desabilitadas em config.js

Criar objeto: clicar sobre o canvas
Criar morfismo: segurando a tecla SHIFT, clicar sobre o objeto de origem, manter o mouse pressionado e soltá-lo sobre o objeto de destino
Criar endomorfismo: segurando a tecla SHIFT, clicar sobre o objeto e soltá-lo sobre o mesmo objeto (se ocorrer um duplo clique em até 200 milissegundos, a endomorfismo do objeto é desfeita)

Selecionar objeto: clicar sobre o objeto (enquanto selecionado, sua cor fica diferente)
Selecionar morfismo: clicar sobre a alça do morfismo (enquanto selecionado, sua cor e de sua alça ficam diferentes)
Selecionar endomorfismo: clicar sobre a alça do endomorfismo
Selecionar múltiplos elementos individualmente: segurando a tecla CTRL, clicar sobre cada elemento
Selecionar múltiplos elementos com retângulo de seleção: clicar sobre o canvas, arrastar o mouse e soltar

Arrastar objeto: clicar no objeto para selecioná-lo, manter o mouse pressionado e movê-lo, soltá-lo para parar de arrastar
Arrastar morfismo: clicar na alça do morfismo para selecioná-la, manter o mouse pressionado e movê-lo, soltá-lo para parar de arrastar
Arrastar endomorfismo: clicar na alça do endomorfismo para selecioná-la, manter o mouse pressionado e movê-lo, soltá-lo para parar de arrastar
Arrastar múltiplos elementos: clicar em algum elemento selecionado, manter o mouse pressionado e movê-lo, soltá-lo para parar de arrastar (ainda não implementado)

Selecionar tudo: CTRL + A
"Deselecionar" tudo: CTRL + SHIFT + A
Copiar elementos selecionados: CTRL + C
Abrir arquivo com estados salvos: CTRL + O (em desenvolvimento)
Criar arquivo com estados salvos: CTRL + P (em desenvolvimento)
Copiar estado atual: CTRL + S
Colar elementos copiados: CTRL + V
Recortar selecionados: CTRL + X
Refazer última ação: Ctrl + Y (em desenvolvimento)
Desfazer última ação: Ctrl + Z (em desenvolvimento)

Deletar selecionados: DEL (em desenvolvimento)

Abrir página de ajuda: F1
Abrir página para desenvolvedores: F2

Para algumas destas ações, um segundo menu com campos específicos se abre
Dependendo da ação, campos podem ser manipulados neste segundo menu antes que suas ações sejam executadas
Neste segundo menu, campos podem voltar ao padrão se o botão RESET for pressionado
Este segundo menu executa as ações se seu botão OK ou a tecla ENTER forem pressionados
Este segundo menu pode ser fechado sem que sua respectiva ação tome efeito se o botão QUIT ou a tecla ESC forem pressionados
O segundo menu só é executado se todos seus campos estiverem preenchidos
Pode-se passar pelos campos que devem ser preenchidos usando a tecla TAB
O valor do campo type para morfismos também pode ser escolhido com os números de 1 a 5

Trocar tipo dos próximos morfismos criados: Q (morfismo, é o padrão), W (monomorfismo), E (epimorfismo), R (monoepimorfismo), T (isomorfismo)
Desenhar livremente: segurando a tecla TAB, clicar e arrastar mouse enquanto quiser desenhar (o cursor mudará de formato enquanto estiver desenhando)
Trocar cor do pincel de desenho: 0 a 9 (1 a 6 para cores do arco-íris e 7 a 0 para 4 tons monocromáticos, padrão é 7)
Aumentar/diminuir tamanho do pincel de desenho: +/- (1 a 72, padrão é 4)
Apagar último desenho: BACKSPACE
Apagar todos desenhos: CTRL + BACKSPACE

Enquanto o canvas estiver fora de foco, um retângulo cinza o preencherá, indicando que é necessário repor o foco sobre ele, isto pode ser feito simplesmente clicando nele
O foco é perdido ao abrir o console, ao clicar e/ou interagir com o console, ao clicar nas barras de endereço e pesquisa do navegador, quando um alert está sendo exibido, etc
Enquanto ALT + TAB estiver sendo pressionado, o foco é perdido até o usuário voltar à página
Atenção: Ao perder o foco, o retângulo de seleção é apagado e o estado da máquina de estados do teclado e mouse vai para "idle" ou "idle with element(s) selected" (o que significa que isto pode ser usado para resetar quando você perceber que um comando não está funcionando)
